# MOVIE CHECKLIST
> # Веб-приложение позволяет вести check-list просмотра фильмов.


# ОСНОВНЫЕ ФУНКЦИИ
> **Ведение** базы фильмов и режиссеров.

> **Оценка** каждого из фильмов.

> **Отслеживание** просмотра фильмов.

# СТЭК ТЕХНОЛОГИЙ
> **Node.js** v16.13.1

> **React** 

> **GraphQL** 

> **MongoDB**: https://www.mongodb.com/atlas/database

> ****
